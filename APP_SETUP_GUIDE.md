# APP SETUP GUIDE

## Step 1: Create Power Apps Canvas App
1. Log in to Power Apps.
2. Click on the "Create" button.
3. Select "Canvas app from blank."
4. Name your app and choose the appropriate format (tablet or phone).

## Step 2: Connect to Excel Data
1. Click on the "Data" option in the left panel.
2. Select "Add data source."
3. Search for and select "Excel Online (Business)."
4. Connect to your Excel file by choosing it from your OneDrive or SharePoint location.
5. Select the required table from your Excel file to use as the data source.

## Step 3: Configure Alerts
1. In your app, navigate to the section where you want to implement alerts.
2. Use Power Automate or the built-in notifications feature.
3. For Power Automate:
   - Click on "Action" in the top menu, then select "Power Automate."
   - Create a new flow that triggers based on data conditions (e.g., when a new value is added to Excel).
   - Configure the flow to send alerts (e.g., emails or push notifications) as needed.
4. Test the alerts by making changes to the Excel data to ensure notifications are sent appropriately.

## Step 4: Save and Publish the App
1. Click on "File" > "Save" to save your changes.
2. Publish the app by going to "File" > "Publish to everyone."
3. Share the app with users who need access to it.
